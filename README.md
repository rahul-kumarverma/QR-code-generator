# QR Code Generator

A simple QR Code Generator built with Python and Tkinter.

## Features

- Generate QR codes
- Save as PNG
- GUI Interface
- Preview generated QR

## Libraries Used

- qrcode
- Pillow
- tkinter

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python qr_generator.py
```
